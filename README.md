# Проект: Место

### Обзор

Учебный проект.
Страница "Место" для третьего спринта. Заготовка для интырнет-галереи. Респонсив HTML/CSS/JS/Webpack.

#### Функционал
- изменение профиля пользователя
- добавление карточек в общую галерею
- удаление своих карточек
- лайки
